---
title: "element vs. attribute"
notetype : feed
date : 23-01-2022
---

```js

<button className="submit"> Submit </button>
👉 button is an element (also called a button tag)
👉 className is an attribute

<App title="Toilet Paper"></App>
👉 여기서 title은 attribute (App is a component) title값은 항상 "Toilet Paper"인 것.
title값이 계속 바뀐다면, assign value(값) to attirbutes 하기 위해서는 : <App title={변수}></App>

```
