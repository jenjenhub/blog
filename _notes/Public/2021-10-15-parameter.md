### 매개변수 vs. 변수

▪️ 매개변수 : 함수에 인자로 쓰이는 변수, '매개변수' ▶️ parameter 

```js
function test (a, b) { 
  } 
```



▪️ 변수 : const, let 으로 생성

```js 
let a =
let b = 
```

이렇게 생성한 a,b는 그냥 '변수'



▪️ 인수 / 전달인자 : 함수에 인자로 들어가는 값  ▶️ argument

```js
function test (3, 4) { 
  }
```

여기서 3, 4 = 인수
